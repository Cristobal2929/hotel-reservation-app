# Hotel Reservation App

A simple hotel reservation system built with **FastAPI**, **SQLAlchemy**, and **Bootstrap 5**.  
The app provides:

- A rich home page with hotel description, amenities and image gallery.
- Room catalog with selection buttons.
- Reservation form with real‑time date validation.
- SQLite persistence (`hotel.db`).
- Email confirmation (SMTP configuration optional).
- Admin dashboard (basic auth) to view, update status and export reservations.
- RESTful API for CRUD operations (admin protected).
